Module_01: 舞蹈類

SYNTH RIDERS
GAME PREVIEW

幻音騎士 (Synth Riders)

在霓虹閃爍的賽道中揮動肢體，捕捉色彩斑斕的節奏寶石。這款作品收錄了 Zedd、Lady Gaga 等頂尖藝術家的曲目，帶來極致的視覺與體感融合。雖然不支援多人連線，但其流暢的打擊感與舞蹈體驗絕對是同類型中的佼佼者。

AUDIO TRIP
GAME PREVIEW

動感音旅 (Audio Trip)

化身為音樂的指揮家，用雙手的節奏球在虛擬空間中揮灑汗水。遊戲充滿賽博龐克感的視覺設計，更支援多人連線模式，讓你能與好友一起在未來的音樂浪潮中恣意共舞。

商店頁面
LES MILLS XR
GAME PREVIEW

LES MILLS XR DANCE

這不僅是舞蹈，更是一場高強度的體能挑戰。將 LES MILLS 專業的健身課程導入 VR，結合沉浸式教練指導與全球流行舞曲，讓你在享受舞蹈樂趣的同時，也能達到驚人的燃脂效果。

商店頁面
SAMBA DE AMIGO
GAME PREVIEW

Samba de Amigo : 虛擬搖搖派對

經典沙鈴派對歡樂回歸！手把化身沙鈴，跟隨節奏球盡情搖擺。遊戲特別加入了 MR 模式，模糊虛實邊界，不論大小朋友都能在繽紛的拉丁與流行樂中感受到最純粹的派對快樂。

Module_02: 鼓類

RAGNAROCK
GAME PREVIEW

Ragnarock

帶領維京戰船出征！揮舞雙槌隨史詩般的金屬與搖滾樂擊碎符文。遊戲鼓組排布類似太鼓達人，打擊感紮實有力，玩起來彷彿化身為戰場中央的鼓手，極具熱血氛圍。

商店頁面
SMASH DRUMS
GAME PREVIEW

Smash Drums

體驗街機般的爵士鼓打擊快感。以豐富的搖滾曲風為核心，其直覺的打擊機制非常適合初學者，是進入 VR 鼓藝世界的絕佳入門選擇。

商店頁面
PARADIDDLE
GAME PREVIEW

Paradiddle

被譽為 VR 爵士鼓遊戲中最具寫實深度的進階之作。不僅具備逼真的錄音回放功能，還能自由配置專屬鼓組，根據真實力道產生動態反饋。無論是自主練習還是即興創作，都能讓你成就真正的鼓手之夢。

PISTOL WHIP
GAME PREVIEW

Pistol Whip

動作片與音遊的完美交響。這是一場結合節奏、槍戰與物理閃避的挑戰。當你的攻擊精準落在節奏拍點上時，爽快感與得分將成倍增長，更支援自定義模組，讓你打造專屬的節奏戰場。
